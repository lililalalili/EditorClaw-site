# RAG 生产规模部署：在线刷题与模拟考试系统

## 使用方法
1. 解压 ZIP 文件。
2. 双击打开 `index.html` 即可离线使用。
3. 也可以把 `index.html` 上传到 GitHub Pages、学校网站、公司内网或任意静态网页服务器。

## 功能
- 练习模式：提交后立即显示正确答案与解析。
- 模拟考试：随机抽取最多 50 题，75 分钟倒计时，交卷后统一评分。
- 错题复习：错题自动保存到浏览器 localStorage。
- 题库浏览：支持章节、题型和关键词筛选，可打印。

## 题型说明
- 单选题：1 个正确答案。
- 多选题：2–4 个正确答案。

## 适用课程
Introduction to Deploying RAG Pipelines for Production at Scale / 生产规模部署 RAG 管线导论。
