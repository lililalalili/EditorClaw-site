# BAAMM 课程在线刷题与模拟考试系统

## 内容
- `index.html`：可直接打开的在线刷题与模拟考试网页系统
- `question_bank.json`：题库数据，共 76 道题
- `BAAMM_题库与详细解析.md`：可阅读、可打印的题库与解析文本

## 题库结构
- 总题量：76
- 单选题：38
- 多选题：38
- 知识点：38
- 设计原则：每个知识点至少二次覆盖，即 1 道单选 + 1 道多选。

## 使用方法
1. 双击打开 `index.html`。
2. 在“逐题刷题”中按模块、题型、知识点筛选练习。
3. 在“模拟考试”中设置题量和考试时间，随机抽题并自动评分。
4. 错题会保存在浏览器本地 localStorage 中，可在“错题本”查看。

## 部署为在线系统
可将整个文件夹上传到 GitHub Pages、校内服务器或任意静态网站托管服务。
如果使用 GitHub Pages：
1. 新建一个 GitHub 仓库。
2. 上传本文件夹内的所有文件。
3. 在仓库 Settings → Pages 中选择部署分支。
4. 访问生成的 GitHub Pages 地址即可在线刷题。

## 注意
本题库根据 NVIDIA DLI “Building AI Agents with Multimodal Models”公开课程大纲和讲义主题整理生成，不包含 NVIDIA 课程内部未公开的原始测验题。
