- `index.html`：统一入口页。
- `RAG_Eval_Study_Guide_NonOfficial.docx`：Word 教材。

## 使用方法

1. 解压整个资料包。
2. 双击根目录下的 `index.html` 打开统一入口。
3. 也可直接打开各子目录下的 `index.html`。
4. 所有网页均为静态文件，可离线使用，也可部署到 GitHub Pages、Netlify、Vercel 或任意静态网页托管服务。
