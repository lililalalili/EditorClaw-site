# NCP-AAI 知识卡片学习系统

本系统根据 NCP-AAI / Agentic AI Professional 中文讲义整理，包含 176 张知识卡片。

## 文件说明

- `index.html`：在线/离线知识卡片学习系统，双击即可打开。
- `cards.json`：结构化卡片数据。
- `cards.csv`：Excel 可打开的卡片表格。
- `anki_flashcards.tsv`：可导入 Anki 的 TSV 文件。

## 卡片结构

正面：术语/缩写  
背面：英文全称、中文全称、中文解释

## 使用方法

1. 解压 ZIP。
2. 双击 `index.html` 使用。
3. 如需在线部署，可上传整个文件夹到 GitHub Pages、Nginx、Apache、NAS Web 服务或任意静态网站空间。

## 功能

- 按章节筛选
- 搜索术语/英文/中文解释
- 点击翻卡
- 熟悉/不熟标记
- 收藏卡片
- 优先复习
- 快速测验
- 本地进度保存
- JSON / CSV / Anki TSV 导出
