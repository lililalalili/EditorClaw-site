# AAI Professional 非官方学习平台

## 打开方式

解压后双击 `index.html`，或把整个文件夹上传到 GitHub Pages、Nginx、NAS、内网服务器等静态网站环境。

## 内容

- `index.html`：统一学习平台入口
- `course_reading.html`：教材网页阅读版
- `quiz/index.html`：在线刷题与模拟考试系统（非官方原创题）
- `flashcards/index.html`：知识卡片系统
- `docs/AAI_Professional_NonOfficial_Textbook.docx`：去官方化改写后的 Word 教材
- `docs/DISCLAIMER.md`：免责声明
- `docs/CITATIONS.md`：引用说明
- `docs/CHANGELOG_DE_RISKING.md`：去侵权风险化改写记录

## 发布建议

发布到公网时，建议保留首页和每个子系统顶部的免责声明，不要使用“出品或授权课程”“出品或授权教材”“出品或授权题库”“授权资料”等表述。
