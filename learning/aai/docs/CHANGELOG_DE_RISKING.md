# 去侵权风险化改写记录

本次统一处理了已生成的教材、题库系统和知识卡片系统，重点修改如下：

## 1. 标题改写

- 将具有官方背书联想的标题统一改为“AAI Professional 非官方学习资料 / 非官方刷题系统 / 非官方知识卡片系统”。
- 弱化“Agentic AI Professional”作为主标题使用，改为“Agentic AI Professional 方向学习资料”。

## 2. 免责声明统一

所有入口页面和子系统顶部均加入统一免责声明，明确说明：

- 不是 NVIDIA 出品或授权的课程、教材、题库或考试材料；
- 未获得 NVIDIA 授权、认证、合作或背书；
- 不包含 NVIDIA DLI 内部课程、官方课件、实验环境、测验题或考试真题；
- NVIDIA、NIM、NeMo、Triton、TensorRT-LLM、DLI 等名称仅用于技术说明和学习引用；
- 请以 NVIDIA 官网、官方文档和考试服务商说明为准。

## 3. 去官方化措辞

- 将“官方考试大纲”“官方认证页面”“出品或授权课程”等表述改为“官网公开考试信息”“公开认证简介”“参考公开课程/认证简介的学习资料”等。
- 将“认证考试完整中文教材”改为“非官方中文学习教材”。
- 将“出品或授权题库/考试系统”类可能造成误解的表述改为“原创练习题库/自测系统”。

## 4. 引用说明

新增 `CITATIONS.md`，说明资料参考的是公开认证简介、公开学习指南、公开技术文档和通用工程实践，不含受限资料。

## 5. 在线平台整合

将教材阅读、题库与模拟考试、知识卡片、Word 教材下载、引用说明和免责声明整合为一个静态网页平台。
