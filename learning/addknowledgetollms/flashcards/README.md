# 《为大语言模型添加知识》缩写知识卡片学习系统

## 文件说明

- `index.html`：在线学习系统主页面，可直接双击打开，也可部署到任意静态网站服务器。
- `flashcards.json`：知识卡片结构化数据。
- `flashcards.csv`：知识卡片表格数据，可用 Excel 打开。
- `README.md`：使用说明。

## 使用方法

1. 解压 ZIP 文件。
2. 双击打开 `index.html`。
3. 在“卡片学习”模式中点击卡片查看反面。
4. 使用“已掌握”和“需复习”记录学习状态。
5. 在“测试模式”中练习缩写与全称的对应关系。
6. 学习进度保存在当前浏览器的 localStorage 中。

## 卡片字段

- 正面：缩写
- 反面：英文全称、中文全称、含义
- 分类：基础概念、数据、评估、模型定制、部署优化、NVIDIA 生态、系统开发等

## 部署说明

该系统为纯前端静态网页，不需要服务器后端。可部署到 GitHub Pages、内网服务器、Nginx、Apache 或任何静态网站托管服务。
