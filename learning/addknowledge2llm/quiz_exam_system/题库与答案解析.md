# 题库与答案解析
## Q001 [单选]  “大语言模型知识增强与模型定制”的核心目标最准确的是哪一项？
模块：M1 课程导论与端到端工作流  难度：易
A. 只通过提示词让模型在任何场景下回答正确
B. 围绕数据、评估、定制和部署优化，系统提升 LLM 在特定场景中的可靠性与适配性
C. 完全替代检索增强生成（RAG）系统
D. 只压缩模型参数以降低显存占用

**正确答案：** B

**解析：** 本课程的主线是从数据准备开始，经模型评估、CPT/SFT/DPO 等定制方法，再到量化、剪枝、蒸馏等部署优化，形成端到端工作流。

## Q002 [单选]  在企业级 LLM 应用中，为什么通用模型通常还需要“添加知识”或定制？
模块：M1 课程导论与端到端工作流  难度：易
A. 因为通用模型不会生成自然语言
B. 因为通用模型可能缺乏最新、专有或领域特定信息，也可能不符合特定任务风格
C. 因为所有通用模型都无法进行推理
D. 因为只要模型够大就不需要任何数据治理

**正确答案：** B

**解析：** 通用模型具有广泛能力，但企业/领域场景往往需要更新知识、专有术语、特定输出格式、风格或安全要求。

B. 先明确业务目标、数据来源、评估指标与可接受风险
C. 先做剪枝以节省成本

**正确答案：** B

**解析：** 明确目标和评价标准是后续数据筛选、训练策略、部署优化的依据，否则容易训练出“看似改进但不可验证”的模型。
