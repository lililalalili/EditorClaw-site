# 大语言模型知识增强与模型定制｜在线题库与模拟考试系统（非官方学习资料）

本文件夹是一个纯静态网页系统，可用于课程自学、题库练习和模拟考试。

## 内容

- 题库总量：120 题
- 单选题：70 题
- 多选题：50 题（每题正确答案 2–4 个）
- 模块：8 个

## 使用方法

1. 解压 ZIP。
2. 双击 `index.html`，即可在浏览器中使用。
3. 若要局域网访问，可在本目录运行：

```bash
python -m http.server 8000
```

然后访问 `http://本机IP:8000/`。

## GitHub Pages 发布

1. 新建 GitHub 仓库。
2. 上传本文件夹全部内容。
3. 在仓库 Settings → Pages 中启用 GitHub Pages。
4. 保持入口文件为 `index.html`。

## 2026-04 修复说明

若本地双击 `index.html` 后出现“按钮点击无效”，通常是浏览器、插件或 `file://` 环境限制了 `localStorage`，导致页面初始化中断。此修复版已完成：

- 为 `localStorage` 增加安全降级：不可用时使用临时内存存储，不影响刷题与考试；
- 将按钮处理改为统一事件委托，动态生成的按钮也可稳定点击；
- 为所有静态按钮增加 `type="button"`；
- 增加初始化异常提示，便于排查文件缺失或浏览器限制问题。

推荐使用方式：
1. 解压整个文件夹后双击 `index.html`；
2. 若仍有浏览器限制，在该文件夹打开命令行运行：`python -m http.server 8000`，然后访问 `http://127.0.0.1:8000/`。
