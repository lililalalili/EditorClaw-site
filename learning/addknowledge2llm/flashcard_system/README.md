# 大语言模型知识增强与模型定制｜在线知识卡片学习系统（非官方版）（非官方学习资料）

## 内容
- 知识卡片：106 张
- 模块：7 个
- 每张卡片包含：术语/缩写、英文全称、中文全称、中文解释
- 支持：翻转学习、搜索、模块筛选、随机小测、学习进度、CSV/JSON 导出

## 使用方法
1. 解压 ZIP。
2. 双击 `index.html` 即可本地使用。
3. 如浏览器对本地文件限制较多，可在该文件夹中运行：

```bash
python -m http.server 8000
```

然后访问：

```text
http://127.0.0.1:8000/
```
