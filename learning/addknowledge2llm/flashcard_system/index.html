使用说明

本系统为单文件静态网页，可直接双击 index.html 打开，也可上传到 GitHub Pages、内网服务器或课程平台。

快捷键：学习卡片时按 Space 翻转，按 ←/→ 切换，按 1 标记已掌握，按 2 标记待复习。

学习进度默认保存在浏览器本地。如果浏览器限制本地存储，系统会自动降级为当前页面会话内存储，不影响按钮点击。

如需更稳定地本地运行，可在文件夹中执行：
python -m http.server 8000
然后访问 http://127.0.0.1:8000/。
