# RAG 智能体课程知识卡片学习系统

## 文件说明

- `index.html`：单文件在线知识卡片学习系统，双击即可本地使用，也可上传到 GitHub Pages / 学校网站 / 内网服务器。
- `rag_agents_knowledge_cards.json`：知识卡片 JSON 数据。
- `rag_agents_knowledge_cards.csv`：知识卡片 CSV 数据，可用 Excel 打开。

## 使用方法

### 本地使用

解压 ZIP 后，直接双击 `index.html`。

### 在线部署

将整个文件夹上传到任意静态网页服务器即可；也可以只上传 `index.html`，因为卡片数据已经嵌入在网页中。

## 功能

- 翻卡学习：正面术语/缩写，反面英文全称、中文全称、中文解释。
- 搜索与模块筛选。
- 已掌握 / 待复习标记，进度保存在浏览器 localStorage。
- 随机复习。
- 术语测试。
- 全部卡片浏览。
- JSON / CSV 导出。
- 打印 / 保存 PDF。
