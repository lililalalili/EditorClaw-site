# RAG 智能体课程题库与在线刷题/模拟考试系统

## 使用方法

1. 直接双击 `index.html` 即可在浏览器中使用。
2. 上传整个文件夹到 GitHub Pages、学校网站、机构内网或任意静态网站服务器即可在线使用。
3. 系统无需后端、无需数据库、无需安装依赖。
4. 答题记录保存在当前浏览器 localStorage 中。

## 功能

- 72 道题库：36 个知识点 × 2 道题。
- 单选题 + 多选题。
- 在线刷题模式：筛选、搜索、即时解析、错题模式。
- 模拟考试模式：24/36/60/72 题随机组卷，交卷后评分。
- 题库与统计：知识点覆盖统计、导出 JSON、导出 CSV、打印题库。

## GitHub Pages 发布

1. 新建 GitHub 仓库。
2. 上传 `index.html`。
3. 打开仓库 Settings → Pages。
4. Source 选择 `Deploy from a branch`，Branch 选择 `main` 和 `/root`。
5. 保存后等待 Pages 生成网址。
