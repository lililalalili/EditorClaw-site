# BAAMM 缩写知识卡片学习系统

## 内容
- `index.html`：在线/离线知识卡片学习系统
- `flashcards.json`：知识卡片数据，共 80 张
- `BAAMM_缩写知识卡片_Anki导入.csv`：可导入 Anki 的卡片表
- `BAAMM_缩写知识卡片.md`：可阅读、可打印的 Markdown 版卡片

## 卡片格式
- 正面：缩写
- 反面：英文全称、中文全称、含义、类别、备注

## 功能
- 卡片翻面学习
- 按类别/备注筛选
- 搜索缩写、英文、中文和含义
- 随机顺序学习
- 标记“已掌握 / 待复习”
- 自测模式：看英文全称、中文全称或含义选择缩写
- 完整列表
- Anki CSV 导出

## 使用方法
双击打开 `index.html` 即可离线使用。
也可以将整个文件夹上传到 GitHub Pages、校内服务器或任意静态网站托管平台。
