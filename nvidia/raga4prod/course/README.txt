# Introduction to Deploying RAG Pipelines for Production at Scale
## 在线课程学习平台

打开 `index.html` 即可使用。该网站为纯前端单文件应用，可离线运行，也可上传到 GitHub Pages、内网服务器或课程平台。

## 功能
- 课程首页与学习路径
- 教材阅读
- 知识卡片
- 在线刷题
- 模拟考试
- 实验手册
- 学习进度
- 章节导航
- 资源下载

## 数据文件
- course_chapters.json：教材章节数据
- flashcards.json：知识卡片
- question_bank.json：题库
- labs.json：实验手册
- production_checklist.csv：生产上线检查表
