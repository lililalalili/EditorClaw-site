# RAG Production Flashcards System

这是《Introduction to Deploying RAG Pipelines for Production at Scale》讲义配套知识卡片学习系统。

## 使用方法

1. 直接双击 `RAG_Production_Flashcards_System.html` 即可在浏览器中离线使用。
2. 也可以将该 HTML 文件重命名为 `index.html` 后上传到 GitHub Pages、内网服务器或课程平台。
3. 学习进度保存在浏览器 localStorage 中，包括：
   - 已掌握
   - 需复习
   - 收藏

## 功能

- 127 张中英双语术语卡片
- 正面：术语/缩写
- 背面：英文全称、中文全称、中文解释
- 章节筛选
- 全文搜索
- 收藏
- 已掌握/需复习
- 随机复习
- 题库浏览
- 键盘快捷键：Space 翻卡，左右方向键切换
