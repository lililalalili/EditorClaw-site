使用说明

  • 直接双击 index.html 即可离线使用；也可以把整个文件夹上传到 GitHub Pages、内网 Nginx、Apache 或任意静态网站空间。
  • “章节刷题”会即时给出答案解析，适合学习；“模拟考试”默认 60 题、120 分钟，适合考前训练。
  • 错题、收藏和正确率保存在当前浏览器的 localStorage，不会上传到服务器。
  • 如需扩展题库，可编辑 question_bank.json，也可在 index.html 中搜索 QUESTION_BANK 替换嵌入数据。

静态部署模板：GitHub Pages

1. 新建 GitHub repository
2. 上传 index.html、question_bank.json、metadata.json
3. Settings → Pages → Deploy from branch → main / root
4. 打开生成的 Pages URL 即可在线访问

内网部署模板：Python 简易服务器

cd NCP_AAI_Online_Quiz_System
python -m http.server 8080
# 浏览器访问：http://你的电脑IP:8080

内网部署模板：Nginx

server {
    listen 80;
    server_name aai-quiz.local;
    root /var/www/NCP_AAI_Online_Quiz_System;
    index index.html;
    location / { try_files $uri $uri/ /index.html; }
}
